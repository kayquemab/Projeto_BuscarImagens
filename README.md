# 📌 BuscarImagens

Aplicação web que permite buscar imagens em tempo real utilizando a API do Unsplash.

## 🚀 Tecnologias

* HTML5
* CSS3
* JavaScript
* Unsplash API

## ✨ Funcionalidades

* Busca de imagens por palavra-chave
* Exibição dinâmica de resultados
* Botão "ver mais" para carregar novas imagens

## ⚙️ Como executar

Baixar ou clonar o repositório e abrir o arquivo `index.html` no navegador

## 📌 Observações

* Projeto estático (não requer instalação)
* Necessário utilizar uma chave da API do Unsplash
* Os resultados podem variar conforme a busca realizada
